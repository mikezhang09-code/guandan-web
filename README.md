# 🃏 扑克牌桌 · Card Table

两个用两副牌玩的经典中式扑克游戏 —— **掼蛋** 和 **升级（拖拉机）**，纯前端实现，一人对三个电脑 AI，浏览器打开即玩。

🎮 **在线试玩**：https://mikezhang09-code.github.io/guandan-web/

## 特点

- **零依赖、纯本地**：单 HTML 文件，原生 JavaScript，无后端、无构建、无需联网。
- **不需要任何 API / Key**：电脑对手是本地规则算法（启发式决策），不调用大模型，免费、离线。
- **完整规则**：级牌、百搭、炸弹、进贡还贡（掼蛋）；叫主、扣底、拖拉机、抓分、翻倍、上台（升级）。
- **图文规则页**：用真实的扑克牌演示每条规则。

## 游戏

### 🀄 掼蛋（`guandan.html`）
2 副牌（108 张），4 人 2 队，比谁先出完牌。

- 牌型：单张、对子、三张、三带二、顺子、三连对（木板）、钢板、炸弹（4~8 张）、同花顺、四王炸
- 级牌被抬高至「王」之下；红桃级牌为**百搭（逢人配）**
- 大小规则：同型比大小、炸弹通杀、炸弹分等级
- **进贡 / 还贡**：双下 / 单下 / 抗贡 / 先出权
- 头游所在队按队友名次升级，先**打过 A** 的一队获胜
- AI 会保结构、不拆炸、配合队友

### 🚜 升级 / 拖拉机（`shengji.html`）
2 副牌（108 张），4 人 2 队，庄家守分、闲家抓分。

- 每人 25 张 + 8 张底牌；庄家**叫主**选主牌花色、拿底牌后**埋底**
- 主牌体系：大王 ＞ 小王 ＞ 主级牌 ＞ 副级牌 ＞ 主花色 A…3
- 牌型：单张 / 对子 / 连对（拖拉机）
- 跟牌：必须跟花色、有对跟对、缺门可垫牌或用主牌**将吃**
- 分牌：5=5、10=10、K=10，全副 200 分；闲家赢最后一墩**扣底翻倍**
- 闲家保 80 分线，按分数**升级 / 上台**换庄，打过 A 守住即获胜

> 说明：升级为清晰起见做了简化 —— 庄家固定轮转并由庄家定主牌花色（未做发牌时抢叫主），拖拉机将吃仅支持同花色连对；其余规则与标准升级一致。

## 本地运行

无需任何环境，直接用浏览器打开 `index.html` 即可。或起一个静态服务器：

```bash
python3 -m http.server 8000
# 打开 http://localhost:8000
```

git add -A && git commit -m "..." && git push

## 操作

- **掼蛋**：点牌选中 → `出牌` / `不出` / `提示` / `理牌`
- **升级**：坐庄时先点花色按钮**叫主**、选 8 张点**埋底**；出牌时选牌按 `出牌`，不确定按 `提示`
- 每个游戏页左下角「🏠 选择游戏」可返回入口

## 开发说明

游戏逻辑全部内联在各自的 HTML 文件中。两个游戏的规则引擎都经过无头模拟测试（AI 自动对战数千手）验证：牌型识别、大小比较、跟牌合法性、牌张守恒、完整对局流程均无误。
